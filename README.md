# S3 Integrated System Solutions — Fire Safety Website

**🔗 Live Demo:** [https://shivangibuilds.github.io/S3ISS/](https://shivangibuilds.github.io/S3ISS/)

First-draft website for the Fire Safety Solutions Division of S3 Integrated System Solutions. Bilingual (English / हिन्दी), single-page, mobile-first.

## What's in here

| File | Purpose |
|---|---|
| `index.html` | Markup — hero, story, 6 service cards, India coverage map, why-us, contact form, footer |
| `styles.css` | Design system — navy/red brand palette, light theme, responsive breakpoints |
| `script.js` | Language toggle, scroll reveals, form validation (name / 10-digit phone / email / city / place) |
| `videos/sprinkler-clip.mp4` | Local video clip for the sprinkler card |

## Sections

1. **Hero** — "Fire doesn't wait. Neither do we."
2. **Story** — "Fire-ready, or not fire-ready?"
3. **Six Solutions** — Hydrants · Sprinklers · Fire Alarms · Audit · Design · NOC & Compliance
4. **Where We Work** — India map with pins on Ranchi (HQ) · Patna · Bengaluru, plus a "+ GROWING" badge
5. **Why Us** — Three reasons
6. **Free Audit form** — strict validation, lead-gen
7. **Footer**

## Notes

- Email is the only public contact (no phone numbers exposed).
- Mailto subject and body are pre-filled and switch between English and Hindi.
- All videos are royalty-free (Mixkit) except the sprinkler clip, which is excerpted from a Cheshire Fire & Rescue Service public demonstration.
- This is a draft for review — content (testimonials, real project photos, certifications) to be supplied by Avinash.
